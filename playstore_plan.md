# SAIVEX Play Store Plan

## Important
The Play Store AAB does not host the SAIVEX Flask backend. The Android app needs a live backend URL or must be rebuilt as a native app.

## Best Path
1. Get original SAIVEX backend hosted.
2. Create Android WebView or native wrapper.
3. Generate signed AAB.
4. Run closed testing.
5. Submit production release.

- 512x512 app icon
- Feature graphic
- App screenshots
- Privacy policy
- App description
